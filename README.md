# Color Flipper
- Simple colors
- Hex colors
- Rainbow colors

### Simple Colors
- 12 colors (Black, White, Red, Green, Blue, Yellow, Cyan, Magenta, Silver, Gray, Maroon, Olive)

### Hex colors
- Random colors (0 - 9, A - F)
 (eg. #DEA267, #89EADA, #E2FCD8, #1AD2CA, #5B2255)

### Rainbow colors
- Rainbow animation with colors (Red, Orange, Yellow, Green, Blue)
- including 5 colored buttons and 'Restart' button to control the animations
